# coffee-list
## 요구사항
- 목표 : 다양한 커피를 보여주는 앱 개발
- 기능: 세부항목은 커피명, 커피특징 및 내용, 커피사진등을 포함하여 내용 구성
            메인화면은 리스트형태로 세부항목 페이지를 프리뷰할 수 있도록 구성
